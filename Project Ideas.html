Project Ideas

CO2 Racer

An exciting competitive project for Tech Mandatory Industrial classes. A project aimed at developing basic hand tool skills and encourage understanding in basic engineering via an engaging project.

RC Pop Pop Boats

A clever and intriging introduction to electronics and engineering inside a compact and fun to run project. Students can create these RC pop pop boats as a kit while retrofitting electronic components to help it improve playability.

Catapult

A project that can be designed with woodwork joinery and combine engineering aspects of force and energy in engineering structures and mechanisms. This project has the adaptability to all year groups determined by complexity and in additional can be gamified for additional engagement and motivation.
